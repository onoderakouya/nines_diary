# ナインズ農業日誌

ナインズ農業日誌は、農作業の現場で使うことを想定した **PHP + SQLite 製の記録アプリ** です。  
日々の作業日誌・出荷・資材費・病害虫の記録を一元管理し、管理者向けの集計・月次レポート作成まで行えます。

## アプリ概要

- 対象: 農場スタッフ（研修生）と管理者
- 目的: 日々の入力を揃えて蓄積し、後から分析・レポートに活用する
- 特徴: モバイル利用を意識したシンプルな入力導線（大きめのボタン、迷いにくい画面遷移）

## 主要機能

- ログイン認証（ユーザー/管理者ロール）
- 日誌入力・一覧
- 出荷入力・一覧・集計・CSVエクスポート
- 資材費入力・一覧・集計・CSVエクスポート
- 病害虫記録・履歴・集計・CSVエクスポート
- 管理者機能
  - 全体集計
  - 月次レポート
  - ユーザー追加
  - パスワードリセット
- PWA 関連ファイル（`manifest.webmanifest`, `sw.js`）を含む構成

## セットアップ手順

### 1. 動作要件

- PHP 8.x（`pdo_sqlite` 拡張を有効化）
- SQLite3

### 2. リポジトリを配置

任意の作業ディレクトリに本リポジトリを配置します。

### 3. データディレクトリを準備

`data/app.sqlite` を配置し、Web サーバー実行ユーザーが読み書きできるようにします。  
（必要に応じて `uploads/` ディレクトリにも書き込み権限を付与してください。）

### 4. 開発サーバー起動

リポジトリ直下で以下を実行します。

```bash
php -S 0.0.0.0:8000
```

### 5. ブラウザでアクセス

- `http://localhost:8000/login.php`

ログイン後、ホーム画面から各機能（日誌/出荷/資材費/病害虫/管理者メニュー）を利用できます。
